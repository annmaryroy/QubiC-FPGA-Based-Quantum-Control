# 1. Windows PowerShell Setup

This section contains the PowerShell commands used to prepare the working environment.

---

## Check PowerShell Version

```powershell
$PSVersionTable
```

---

## Verify Git Installation

```powershell
git --version
```

---

## Verify Python Installation

```powershell
python --version
```

---

## Verify pip Installation

```powershell
pip --version
```

---

## Create Workspace

```powershell
cd D:\
mkdir software
cd software
```

Your workspace is now ready.
