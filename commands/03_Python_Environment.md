# 3. Create the Python Environment

Navigate to the workspace.

```powershell
cd D:\software
```

Create the virtual environment.

```powershell
python -m venv qubic_env
```

Activate it.

```powershell
qubic_env\Scripts\activate
```

Upgrade pip.

```powershell
python -m pip install --upgrade pip
```

Verify.

```powershell
python --version
```

```powershell
pip --version
```
