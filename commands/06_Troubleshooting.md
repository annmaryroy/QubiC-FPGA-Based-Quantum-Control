# 6. Troubleshooting

This page documents common issues encountered during the installation and execution of the QubiC framework on Windows.

## Git clone failed

Possible causes:

- Internet connection unavailable
- Firewall restrictions
- Proxy configuration
- GitLab temporarily unavailable

---

## Python package installation failed

Possible causes:

- Incorrect Python version
- Missing virtual environment activation
- Package dependency conflicts

---

## Jupyter Notebook does not start

Verify:

```powershell
jupyter notebook
```

If Jupyter is not found, install it using pip.
