# 4. Install Python Dependencies

Activate the environment.

```powershell
qubic_env\Scripts\activate
```

Install the required packages.

```powershell
pip install numpy==1.26.4
```

```powershell
pip install qutip==5.0.4
```

> More packages will be added based on the working environment used during the internship.

Verify:

```powershell
pip list
```
