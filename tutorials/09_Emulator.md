# Tutorial 9 – Emulator

## Objective

Learn how to verify pulse-level programs without FPGA hardware.

---

## Topics Covered

- Emulator
- Pulse simulation
- Waveform visualization
- Software verification

---

## Workflow

1. Compile the pulse program.
2. Execute it in the emulator.
3. Inspect generated waveforms.
4. Verify correctness before hardware execution.

---

## Importance

The emulator allows software development and debugging without access to physical RFSoC hardware.
