# Tutorial 2 – Multiple Qubits + Gates

## Objective

Understand how QubiC performs operations involving multiple qubits and quantum gates.

---

## Topics Covered

- Multiple qubits
- Gate scheduling
- Parallel execution
- Pulse synchronization

---

## Workflow

1. Load multiple qubits.
2. Define gate operations.
3. Compile the pulse schedule.
4. Execute.
5. Observe the generated pulse sequence.

---

## Key Concepts

- Single-qubit gates
- Two-qubit gates
- Synchronization
- Timing

---

## Expected Output

Pulse schedules for multiple qubits executing quantum gate operations.
