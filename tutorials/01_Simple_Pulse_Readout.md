# Tutorial 1 – Simple Pulse + Readout

## Objective

Learn how to generate a microwave control pulse and perform qubit readout using the QubiC framework.

---

## Prerequisites

- QubiC repositories cloned
- Python virtual environment activated
- Required Python packages installed
- Jupyter Notebook running

---

## Notebook

Official notebook:

```
simple_pulse_readout.ipynb
```

---

## What You Will Learn

- Pulse generation
- Pulse scheduling
- Readout configuration
- Measurement workflow

---

## Workflow

1. Import the required Python modules.
2. Load the qubit configuration.
3. Generate the control pulse.
4. Configure the readout pulse.
5. Execute the sequence.
6. Observe the acquired data.

---

## Key Concepts

- Microwave pulse
- Pulse envelope
- DAC
- ADC
- Readout resonator

---

## Expected Output

The notebook generates a pulse sequence and acquires the corresponding readout signal.

---

## Observations

This tutorial provides the simplest example of pulse-level programming in QubiC and serves as the foundation for subsequent tutorials.
