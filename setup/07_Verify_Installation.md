# 7. Verify the Installation

## Objective

Before executing the official **LBL QubiC** tutorial notebooks, verify that the software environment has been configured correctly and that all required tools are accessible from **Windows PowerShell**.

---

## Verify Git Installation

Confirm that Git is installed and accessible.

```powershell
git --version
```

---

## Verify Python Installation

Check that the correct Python version is available.

```powershell
python --version
```

Expected version:

```text
Python 3.10.x
```

---

## Verify pip Installation

Ensure that the Python package manager is installed.

```powershell
pip --version
```

---

## Verify Installed Packages

Display the list of installed Python packages.

```powershell
pip list
```

Verify that the required QubiC dependencies have been installed successfully.

---

## Verify Jupyter Notebook

Launch Jupyter Notebook.

```powershell
jupyter notebook
```

If Jupyter Notebook starts successfully and opens in your web browser, the Python environment has been configured correctly.

---

## Installation Checklist

Before proceeding, confirm that the following requirements have been met:

- ✅ Git is installed.
- ✅ Python 3.10 is installed.
- ✅ pip is available.
- ✅ Python virtual environment is created and activated.
- ✅ Required Python packages are installed.
- ✅ Jupyter Notebook launches successfully.
- ✅ Official QubiC repositories have been cloned.
