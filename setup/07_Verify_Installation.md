# 7. Verify the Installation

Before running experiments, verify that the software environment has been configured correctly.

---

## Verify Git

```powershell
git --version
```

---

## Verify Python

```powershell
python --version
```

---

## Verify pip

```powershell
pip --version
```

---

## Verify Installed Packages

```powershell
pip list
```

---

## Verify Jupyter

```powershell
jupyter notebook
```

If Jupyter starts successfully, the installation is complete.

---

You are now ready to execute the official QubiC tutorial notebooks.
