# 6. Run Jupyter Notebook

The official QubiC tutorials are provided as Jupyter notebooks.

---

## Start Jupyter

After activating the virtual environment, run:

```powershell
jupyter notebook
```

A browser window should open automatically.

If it does not, copy and paste the URL displayed in the PowerShell window into your browser.

---

## Open the Tutorial

Navigate to the required notebook.

Example:

```
QubicTutorial/
```

Open the notebook you want to execute.

---

## Execute

Run the notebook cells sequentially from top to bottom.
