# 2. Installing Git

Git is used to download the official QubiC repositories from GitLab.

## Download

https://git-scm.com/downloads

Install Git using the default settings.

## Verify Installation

Open Windows PowerShell and run:

```powershell
git --version
```

Example output:

```
git version 2.xx.x.windows.x
```

If the version is displayed, Git has been installed successfully.
