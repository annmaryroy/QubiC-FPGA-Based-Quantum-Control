# 2. Install Git

## Objective

Git is a distributed version control system used to download and manage the official **LBL QubiC** repositories hosted on GitLab. It is required to clone the QubiC software and related repositories used throughout this guide.

---

## Download Git

Download the latest version of Git for Windows from the official Git website:

🔗 **Git Downloads:** https://git-scm.com/downloads

or using Markdown:

[Download Git for Windows](https://git-scm.com/downloads)

Install Git using the **default installation settings**.

---

## Verify the Installation

After installation, open **Windows PowerShell** and run:

```powershell
git --version
```

---

## Expected Output

The command should display the installed Git version.

Example:

```text
git version 2.xx.x.windows.x
```

If the version number is displayed without any errors, Git has been installed successfully.

---

## Why Git is Required

Git is required to:

- Clone the official QubiC repositories from GitLab.
- Download future updates from the official repositories.
- Maintain version control during development.
