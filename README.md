# QubiC-FPGA-Based-Quantum-Control
Setting up and running the LBL QubiC framework on Windows using PowerShell.


This repository documents my learning journey during my internship at **National Institute of Technology Calicut (NITC)** under **Dr. Nikhil Kumar C S**.

The goal of this repository is to help students and researchers install, configure, and run the official QubiC framework from scratch on Windows.

---

## What you'll learn

- Installing the required software
- Cloning the official QubiC repositories
- Creating a Python virtual environment
- Installing dependencies
- Running the official tutorial notebooks
- Understanding each tutorial
- Common errors and troubleshooting
